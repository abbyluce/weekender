## Introduction
  - This application helps users find weekend trips from Denver and provides itineraries. I researched and created my own API which was utilized to build the front end application in React. 

## Technologies
  - React
  - Express
  - Router
  - Cypress
  - Heroku
  - React Scroll
  - JavaScript
  - HTML
  - CSS
